Objective: 7.C Understand linked list insertion

Question 4
The following function inserts a node into a doubly linked list.
Assume that the variable, head, is a pointer to the first node in the list.
The member, prev, is a pointer to the previous node, and next is a pointer to the next node.
Consider the following pseudocode.
function insert(head, newNode):
    if head is null:
        head = newNode
    else:
        current = head
        while current.next is not null:
            current = current.next
        current.next = newNode
        newNode.prev = current

What is the time complexity of this function?
A. O(1)
B. O(n)
C. O(n^2)
D. O(log n)
E. Cannot be determined

Answer B
